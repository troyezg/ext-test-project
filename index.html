Как работает сервис?

  • Участники акции регистрируют свои чеки на вашем сайте
  • Сайт передает информацию о чеке по API в сервис проверки чеков
  • Сервис проверки чеков получает состав чека в ФНС
  • Сервис отмечает подходящие позиции в чеке и возвращает данные по API на ваш сайт

Стоимость

Оплата сервиса происходит в формате ежемесячных платежей за каждый активный проект

Подписка включает:

  • Настройку одного проекта
  • Обработку до 1000 чеков в сутки в одном проекте
  • Техническую поддержку по любым вопросам

50 000 руб / мес

*Если количество обработанных чеков за сутки превысило 1000 шт, то за каждый последующий чек вы платите 1,5 руб.

Специальная цена для партнеров

Приглашаем к сотрудничеству рекламные и digital агентства. Партнер получает скидку до 40% на сервис проверки чеков, для проектов своих клиентов.

API

Сервис проверки чеков обменивается данными с промо сайтом через API-интерфейс, который содержит список необходимых методов для приема чеков в виде фото или строки (декодированного QR кода), а также описание параметров для обратного вызова и передачи результатов проверки.
